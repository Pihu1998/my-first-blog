# my-first-blog

*Django Application*

This is a Quora-like Web Blog built using Python, Django, HTML, CSS, BootStrap and JavaScript.

The features includes:

1. Creating Blogs.
2. Reading Blogs
3. Modifying Blogs.
4. The Date and Time of creation of blogs are automatically generated and shown.
